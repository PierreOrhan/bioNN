smallNetworkSimul module¶

This module provide script for simulation of small network. By defining a simple mask, one can run the ODE solver on a various number of parameter and try to find the function linking them to the equilibrium for one output species.

neuralNetworkSimul¶

Script for obtaining dynamic of small neural network Here we proceed to the rendering of the function: output(X1,X2) for X1 activator and X2 inhibitor (killer template) Equations are solved in parallel on the CPU

ecoEvoSimul¶

Script file to study the evolution of the standard 2-species predator-prey system.
